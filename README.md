# 方块矿境

一款采用原创像素矿物风格制作的 Android 下落方块游戏。

## 游戏功能

- 标准下落方块玩法
- 左右移动与旋转
- 加速下降和快速落地
- 幽灵落点提示
- 消行、计分和等级系统
- 本地保存最高分
- 暂停、继续和重新开始
- 七种原创像素矿物材质
- 完全离线
- 无广告
- 无联网权限

## APK 下载

请前往本仓库的 Releases 页面下载最新版 APK。

## 操作方式

- 左右按钮：移动方块
- 旋转按钮：旋转方块
- 向下按钮：加速下降
- 快速落下按钮：立即落到底部
- 暂停按钮：暂停或继续游戏
