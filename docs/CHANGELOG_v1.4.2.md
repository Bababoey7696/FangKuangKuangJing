# 方块矿境 v1.4.2

## 触控

- 新增“经典 v1.2”和“稳健 v1.4.1”两套可切换触控方案。
- 经典方案恢复 v1.2 左右连移参数。
- 稳健方案延后首次连移，降低短按误走两格。
- 向下软降维持原有快速响应。

## 音频

- 核对确认 v1.2 与 v1.4 实际播放的四首背景音乐文件完全相同，因此保留全部四首，不添加无效重复曲目。
- 播放端轻度衰减 7kHz 以上频段并略微提升中高频，削弱类似雨声的底噪，同时保持空灵感。
- 快速落地改用更短、更清脆的音色，并抑制 140ms 内紧随其后的普通锁定音，避免两声叠加成闷重尾音。
- 震动反馈保持 v1.4.1，不作修改。

## 兼容性

- versionName：1.4.2
- versionCode：9
- 包名与 SharedPreferences 存档键保持兼容。
- 清理临时构建材料后，使用仓库中的正式源码重新执行 Debug 与 unsigned Release 构建检查。
