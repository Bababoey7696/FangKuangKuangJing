# v1.4.1 触控修复说明

## 问题

左右移动键的首次自动连发延迟过短，普通短按可能在抬手前触发第二次移动。

## 修复目标

- 轻点左右键只移动一格。
- 明确按住后才开始连续移动。
- 软降保持原有快速响应。
- 低、标准、高灵敏度仍控制长按连移速度。

## 新的水平移动参数

| 灵敏度 | 首次连发等待 | 后续连发间隔 |
|---|---:|---:|
| 低 | 380ms | 120ms |
| 标准 | 320ms | 94ms |
| 高 | 260ms | 72ms |

修复完成后需通过 Android CI 的 Debug 与 unsigned Release 构建检查。
